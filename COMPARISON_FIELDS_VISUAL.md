## Grid Score Formula

```
Grid Score = (
  connectivity_score + 
  amenities_score + 
  (growth_score × 10) + 
  (investment_score × 10)
) ÷ 4
```

**Example:**
- Connectivity Score: 8.0
- Amenities Score: 7.5
- Growth Score: 0.75 → 7.5 (×10)
- Investment Score: 0.80 → 8.0 (×10)

Grid Score = (8.0 + 7.5 + 7.5 + 8.0) ÷ 4 = **7.75**

## Color Coding (Desktop Table)

- 🟢 **Green highlight** = Best value (highest for positive metrics)
- 🔴 **Red highlight** = Worst value (lowest for positive metrics)
- ⚪ **No highlight** = Middle values or all same

Applied to:
- Price / sqft (lower is better)
- Base Price (lower is better)
- Area (sqft) (higher is better)
- Grid Score (higher is better)

## Responsive Breakpoints

- **Desktop:** ≥768px → Side-by-side table view
- **Mobile:** <768px → Swipeable card view

## User Interactions

### Desktop
- ✕ Remove button on each property column
- 📥 Export button (PDF or CSV)
- ✕ Close modal button
- Click outside to close

### Mobile
- ← → Swipe to navigate between properties
- Tap dots to jump to specific property
- Remove from Compare button on each card
- 📥 Export button (PDF or CSV)
- ✕ Close modal button

## Loading States

1. **Initial load:** Spinner with "Loading comparison data..."
2. **Amenities fetch:** "Loading..." → "School, Mall..." or "N/A"
3. **Error state:** "Failed to Load Comparison" with Retry button

## Empty States

1. **No properties:** "No Properties to Compare"
2. **Less than 2:** "Please select at least 2 properties to compare"
3. **Missing data:** "N/A" for individual fields
