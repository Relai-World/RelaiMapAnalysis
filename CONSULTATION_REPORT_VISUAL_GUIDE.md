## 🎨 Color Scheme

### Primary Colors
- **Blue**: `#3350c0` - Headers, buttons, accents
- **Green**: `#10b981` - Success, highlights
- **Yellow**: `#f59e0b` - Warnings, concerns
- **Gray**: `#6b7280` - Secondary text

### Backgrounds
- **White**: `#ffffff` - Main background
- **Light Gray**: `#f9fafb` - Cards, sections
- **Light Blue**: `#ecfdf5` - Highlights
- **Light Yellow**: `#fffbeb` - Expert notes

### Gradients
- **Stat Cards**: Purple gradient (`#667eea` to `#764ba2`)

## 💡 Visual Feedback

### Map Capture
```
Before Click:
┌──────────┐
│ 📸 Capture│  ← White background
│ Map View │     Blue border
└──────────┘

During Click:
┌──────────┐
│ 📸 Capture│  ← Green background
│ Map View │     White text
└──────────┘     (1 second)

After Click:
┌──────────┐
│ 📸 Capture│  ← Back to white
│ Map View │     + Notification
└──────────┘
```

## 🎯 Key Interactions

### 1. Hover States
- **Buttons**: Slight lift + shadow
- **Cards**: Border color change
- **Links**: Underline appears

### 2. Click States
- **Buttons**: Press down effect
- **Checkboxes**: Smooth transition
- **Inputs**: Border color change

### 3. Loading States
- **PDF Generation**: Progress notification
- **Map Capture**: Button color change
- **Data Loading**: Spinner animation
