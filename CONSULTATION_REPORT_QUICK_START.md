# Consultation Report - Quick Start

## 🚀 How to Use

### During Consultation

1. **📸 Capture Important Map Views**
   - Click the **"📸 Capture Map View"** button (top-right of map)
   - Capture different areas with relevant layers enabled
   - Each capture is saved automatically

2. **🏠 Browse Properties**
   - Search and view properties
   - System automatically tracks what you view
   - Add properties to comparison

3. **📊 Compare Properties**
   - Select 2+ properties
   - Click "Compare" button
   - Review comparison table

### After Consultation

4. **📋 Generate Report**
   - Click **"📥 Export"** in comparison modal
   - Select **"📋 Consultation Report"**
   - Fill in:
     - Client name (optional)
     - Your name (optional)
     - Expert recommendations
   - Click **"Generate Report"**

5. **📄 Download & Share**
   - PDF downloads automatically
   - Share with client via email/WhatsApp
   - Professional multi-page report ready!

## 📋 What's Included in Report

### ✅ Cover Page
- Session summary
- Client & consultant info
- Locations explored
- Properties viewed

### ✅ Map Screenshots
- All captured map views
- With layers and timestamps

### ✅ Property Comparison
- Full comparison table
- All sections and reviews

### ✅ Location Insights
- Grid scores for each area
- Highlights and concerns

### ✅ Expert Recommendations
- Your personalized advice
- Next steps for client

## 💡 Pro Tips

1. **Capture Early & Often**
   - Capture map views as you discuss different areas
   - Show metro connectivity, flood zones, amenities

2. **Enable Relevant Layers**
   - Turn on layers before capturing (Metro, Floods, etc.)
   - Layers are listed in the report

3. **Write Helpful Recommendations**
   - Be specific about which property you recommend
   - Mention pros/cons
   - Give clear next steps

4. **Track Everything**
   - System auto-tracks locations and properties
   - More data = more comprehensive report

## 🎯 Quick Checklist

Before generating report:
- [ ] Captured 2-3 map views
- [ ] Compared 2-3 properties
- [ ] Prepared expert recommendations
- [ ] Have client name (optional)

## 🔧 Troubleshooting

**Map capture not working?**
- Wait for map to fully load
- Ensure tiles are visible
- Refresh page if needed

**Report missing sections?**
- Capture at least 1 map view
- Compare at least 2 properties
- Add expert notes for recommendations section

**PDF not downloading?**
- Check browser allows downloads
- Try different browser (Chrome recommended)
- Check console for errors (F12)
